 Sales Analysis Dashboard (Power BI)

A simple Power BI dashboard that analyzes sales performance using KPIs and interactive visuals.

 Features:
- KPIs: Total Sales, Total Cost, Total Quantity
- Sales by Channel (Store, Online, Reseller, Catalog)
- Sales by Continent
- Quantity by City
- Sales by Brand
- Slicers for easy filtering

 Tools Used:
- Excel
- Microsoft Power BI
- DAX Measures & Data Modeling
